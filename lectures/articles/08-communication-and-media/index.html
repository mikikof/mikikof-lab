PRACTICE 1 / 学習ノート p.18

「相互理解」までの 5 段階

空欄をタップして、コミュニケーションが成立する条件を順に確認します。物理的な伝達 → 相互理解までの階段です。

私たちは、日頃、同級生や家族と直接話をしたり、SNS やメッセージアプリなどを利用したりして、① をはかっている。
① が成立している状態とは、単に物理的な ② や情報の ③ ができるだけでなく、情報の ④ を通して、発信者と受信者の間で ⑤ ができていることを意味する。
階段の構造: 通信(電波が届く) → 伝達(言葉が届く) → 共有(同じ情報を持つ) → 相互理解(意味が重なる)。「届いた」だけではコミュニケーションとは言わない、というのが POINT 08 の核です。

SUMMARY / POINT 08 まとめ

POINT 08 を 1 枚で俯瞰

「相互理解」を中心に、形態の 3 軸と、変遷の 4 方向に整理し直す。

コミュニケーションとメディア
節 1

定義

「相互理解」が成立した状態。通信や伝達だけでは足りない。

節 2

形態の 3 軸

人数(個別 / マス / 逆マス / 会議)× 位置関係(直接 / 間接)× 同期性(同期 / 非同期)。

節 3

変遷の 4 方向

伝播性・即時性・利便性・情報量。これらの向上が情報社会を形作る。

参考: メディアの定義は「中間にあるもの」
メディア(media)はラテン語の medius(中間)に由来する複数形。発信者と受信者の あいだ に立って、情報を運ぶものすべてを指す。手紙も電話もテレビも SNS もメディア。技術が変わるたびに「中間にあるもの」の性質(伝播性・即時性・利便性・情報量)が変わり、結果としてコミュニケーションの形態自体が変化していく — これが POINT 08 全体の見取り図です。
