          情報とメディアの体系

          この単元では、まず情報そのもの(データ→情報→知識)の性質を押さえ、次にそれを運ぶメディアの3分類を理解する。最後にインターネット時代特有のリテラシーと現象(炎上・エコーチェンバー・フィルターバブル)へと広がる。

          残存性 / 複製性 / 伝播性

          情報は形のない「こと」の世界に属し、もの(物質)とは違う3つの性質をもつ。これらが組み合わさることで、よい情報も悪い情報も拡散しやすくなる。

          残存性
          Persistence
          消えずに残る
          情報は与えてもなくならず、人の記憶や記録媒体に残り続ける
          例過去の SNS 投稿、噂、誤情報がいつまでも残る
          複製性
          Replicability
          簡単に複製できる
          デジタルデータは劣化なく無限にコピーが作れる
          例映画・音楽の違法コピー、文書の流出
          伝播性
          Diffusivity
          短時間に広く伝わる
          情報通信技術の発達で、伝達の速さと範囲が飛躍的に増大
          例コンピュータウイルスの世界規模の被害
          表現 / 伝達 / 記録

          「メディア」は単一の概念ではない。情報を どのように表現するか(表現メディア)、どのように運ぶか(伝達メディア)、どこに保存するか(記録メディア)の3つの役割で分けて捉える。

          表現のためのメディア
          Expression
          文字 音声 図・表
          静止画 動画
          何で表すか

          情報を人間が知覚できる形に変換する。文字は作成・修正が容易、動画は変化を時間軸で伝えられる。

          伝達のためのメディア
          Transmission
          空気 光
          電線 電波
          何で運ぶか

          情報を発信者から受信者へ届ける物理的な経路。電波・電線によって流通範囲が一気に広がった。

          記録のためのメディア
          Recording
          紙 光ディスク
          フラッシュメモリ
          何に保存するか

          情報を蓄えておく媒体。紙はアナログ、光ディスクやフラッシュメモリはデジタルでの長期保存に向く。

          受信の能力 と 発信の能力

          メディアリテラシーは「受け取った情報を見抜く力」だけではない。受信側(分析・評価・判断)と 発信側(適切に表現)の両面を含む。インターネット時代は誰もが発信者になり得るため、両方が必要になる。

          受信側の能力
          分析・評価・判断
          • 多様な視点で情報を分析する
          • 情報の信憑性を評価する
          • クロスチェックで真偽を確かめる

          マスメディアからの情報も含め、すべての情報の受け手に必要。SNSを使わない人にも欠かせない。

          発信側の能力
          表現・伝達
          • メディアの特性を踏まえて表現する
          • 受信者・場面・目的を考慮する
          • 意図を正しく伝える表現を選ぶ

          スマホ普及で誰もが発信者になった現代、適切に発信する能力もメディアリテラシーに含まれる。

          炎上 / エコーチェンバー / フィルターバブル

          SNSの仕組みやアルゴリズムは便利な反面、特有の偏りを生み出す。3つの現象は混同されがちだが、起こる仕組みは別。

          炎上
          批判の殺到
          SNSの投稿などに対して、批判や誹謗中傷の投稿が短時間に集中する現象。発信内容の不適切さが引き金になることが多い。
          エコーチェンバー
          賛同のみが返ってくる
          自分と似た考えの人たちとフォローし合うため、投稿に対して賛同の意見ばかりが返ってくる状況。閉じた小部屋で音が反響する様子に喩えられる。
          フィルターバブル
          偏った情報に包まれる
          検索エンジンや SNS のアルゴリズムが、過去の閲覧履歴に基づき興味に合いそうな情報を優先表示するため、自分の価値観に沿った情報ばかりに包まれる状況。
          違いの要点: エコーチェンバーは「人とのつながり」が原因、フィルターバブルは「アルゴリズム」が原因。両方が同時に働くと、自分の意見を疑う機会が極端に減る。
          クロスチェックという手立て

          悪意をもって虚偽の情報を事実のように見せかけたものをフェイクニュースという。情報の残存性・複製性・伝播性と相性が悪く、長期間・広範囲に影響を残しがち。見抜く基本動作は「複数の情報源で突き合わせる」こと。

          ①情報源を確認する誰が・いつ・どこで発信したかを必ず見る。匿名・出所不明は信頼度が下がる。
          ②複数の情報源で突き合わせる(クロスチェック)1つの情報源だけで判断しない。報道機関・公的機関の発表・専門家の見解を比較する。
          ③感情的に反応する前に立ち止まる怒り・驚き・正義感を強く煽る情報ほど、検証なしの拡散は危険。
          ④広めるかどうかを判断する確証がない情報を広めることは、自分が拡散の一翼を担うことになる。
          × インターネットに上げた情報は、一度拡散すると完全には削除できない。残存性ゆえに、誤情報の影響は長期に残り得る。

          メディアが及ぼす影響

          次の文章中の空欄に適当な語句を入れよ。

          インターネット上では、真実に見せかけた虚偽の情報である①が流布しており、その真偽を見分けるために、複数の情報源から提供された情報を比較する②を行うことが重要となる。
          また、SNS では自分の考えと近いユーザどうしがフォローし合う傾向があるため、自分が投稿した意見に対して、賛同の意見のみが返ってくる③が起きやすく、さらに検索結果なども自分の価値観に沿った情報ばかりが表示されることで、④と呼ばれる現象が起きやすい。
          語群(4語) — タップで空欄に入れる
          ✓ 正答: ① フェイクニュース / ② クロスチェック / ③ エコーチェンバー / ④ フィルターバブル
          ベストフィット
          インターネットや SNS は、メディアが及ぼす影響について理解して利用する必要がある。
          解説

          インターネット上には、フェイクニュースと呼ばれる虚偽の情報が流通しており、その真偽を見極めるためには、情報源の確認やクロスチェックが欠かせない。また、SNS ではユーザが自分の考えと近いアカウントをフォローする傾向があるため、自分の意見に対して賛同の意見のみが返ってくるエコーチェンバーが起きやすく、また検索エンジンのアルゴリズムなどによって、自分の価値観に沿った情報ばかりが表示され、フィルターバブルと呼ばれる状況にも陥りやすい。

          エコーチェンバーとフィルターバブルの違い(補足)

          類似した現象だが、起きる仕組みが違う。混同しないように整理しておく。

          エコーチェンバー:SNS 上で自分と似た考えの人と相互フォローすることで、賛同の意見だけが返ってくる状況。原因は「人間関係の偏り」。閉じた小部屋で音が反響する現象に喩えられる。

          フィルターバブル:検索エンジンや SNS のアルゴリズムが、利用者の閲覧履歴・検索履歴に基づいて興味に合いそうな情報を優先表示するため、結果的に偏った情報に包まれる状況。原因は「アルゴリズムによる選別」。

          両者は同時に働きやすく、自分の意見を疑う機会を奪う。クロスチェックは、この閉じた状態を破る基本動作。

          ⑸ 走行中の自転車への注意や危険を知らせる看板などには、文章を用いたほうがよい。

          解説
          • ⑴ × 適当でない。聴覚に障がいがある人には音声による情報が伝わりにくいため、それぞれの表現メディアの特性だけでなく、さまざまな受信者、条件などを考慮して情報を伝達しなければならない。
