## 凍結リファレンスとは

## 第 1 回制作で確立した重要な技術指針

凍結リファレンスを書き上げる過程で、以下の試行錯誤と確立がありました。これらは [`../SKILL.md`](../SKILL.md) §4-0 に「絶対禁止事項」として記録されています:

1. **SVG 内に KaTeX 構文を書かない**(KaTeX auto-render が誤処理して文字消失)
2. **SVG 内に Unicode subscript/superscript を書かない**(フォントが対応せず化ける)
3. **`<foreignObject>` + KaTeX を使わない**(座標とサイズが噛み合わず重なる)
4. **HTML overlay over SVG を使わない**(% 座標と viewBox のズレ)
5. **番号バッジ(丸+数字)を使わない**(図中の点と紛らわしい)

代わりに以下を採用:
- **記号ラベル方式**: 図中は短い記号(P, n, ∇F)、凡例で「記号 → 数式」の対応
- **凡例方式**: 図(SVG)と説明(HTML+KaTeX)を物理的に分離
- **stack 方式**: 段階を追う図は縦積み(各 row に小 SVG + 短いテキスト)
- **タイムライン方式**: 純 HTML+CSS で縦タイムラインカード
